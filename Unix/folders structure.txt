### Folders structure:
1. /bin		- Essential binaries for system boot and repair.
2. /etc		- System-wide configuration files.
3. /home	- User home directories.
4. /lib		- Shared libraries for essential binaries.
5. /opt		- Optional software packages.
6. /root	- Home directory for the root user.
7. /sbin	- System binaries (superuser only).
8. /usr		- User-related programs and data.
9. /var		- Variable data like logs and temporary files.
10. /proc	- Virtual filesystem providing kernel and process information.
